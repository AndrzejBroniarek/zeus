Καλώς ήλθατε στο "ηλεκτρονικό παραβάν" του συστήματος ψηφοφορίας "Ζευς".

Φτάνοντας σε αυτή τη σελίδα σημαίνει ότι έχετε ακολουθήσει το σύνδεσμο που σας έχει σταλεί στην ηλεκτρονική σας διεύθυνση και έχετε την δυνατότητα να ψηφίσετε για τις "{$T.election.name}".

Για να αποστείλετε την ηλεκτρονική σας ψήφο θα πρέπει να ακολουθήσετε τα επόμενα βήματα. Στο πάνω μέρος της σελίδας θα σας εμφανίζονται επιπλέον οδηγίες χρήσιμες για την ολοκλήρωση του εκάστοτε βήματος.

  1. Συμπλήρωση ψηφοδελτίου. Στο βήμα αυτό θα πρέπει να επιλέξετε την σειρά των υποψηφίων που θέλετε να ψηφίσετε.
  2. Κρυπτογράφηση επιλογής και υποβολή ψήφου. Η επιλογή που κάνατε στο προηγούμενο βήμα θα κρυπτογραφηθεί από "πρόγραμμα πλοήγησης" που χρησιμοποιείτε. Ένα μοναδικό αναγνωριστικό θα σας δοθεί για να μπορέσετε αργότερα να επιβεβαιώσετε οτι η ψήφος σας έχει καταμετρηθεί.

Οποιαδήποτε στιγμή μπορείτε να επισκεφθείτε το σύνδεσμο που σας έχει αποσταλεί και να ξανάψηφίσετε. Κάθε φορά που ψηφίζεται η προηγούμενη ψήφος σας αποκλείετε από το σύνολο των ψήφων που θα καταμετρηθούν για την εξαγωγή του τελικού αποτελέσματος.
